レーダースクリーンに銘柄を追加する
----

### 銘柄コードで追加

{{< image src="img-001.gif" >}}

レーダースクリーンのセルを選択した状態で銘柄コードを入力して <kbd>Enter</kbd> を押すだけで、次々と銘柄を追加できます。

登録された銘柄を削除するには、対象の銘柄のセルを選択した状態で <kbd>Delete</kbd> キーを押します。
<kbd>Ctrl+A</kbd> で全銘柄を選択した状態で <kbd>Delete</kbd> キーを押せば、全銘柄を一度に削除できます。

### 銘柄名で検索して追加

銘柄コードがわからない場合は、銘柄名で検索して追加できます。

1. レーダースクリーンを<kbd>右クリック</kbd> → <kbd><samp>銘柄コードを挿入</samp></kbd> を選択。あるいは <kbd>Alt+S</kbd> を入力
2. <kbd><samp>検索</samp></kbd> ボタンをクリック。あるいは <kbd>Alt+L</kbd> を入力
3. 「株式」タブで銘柄名を入力して <kbd>Enter</kbd>
4. 見つかった銘柄を選択した状態で <kbd><samp>OK</samp></kbd> ボタンをクリック


レーダースクリーンの銘柄をドラッグ＆ドロップで移動する
----

{{< image src="img-002.gif" >}}

登録した銘柄の位置（行）をドラッグ＆ドロップで移動するには、次のようにします。

1. 移動する銘柄の行番号をクリックして選択状態にする（ドラッグで複数の銘柄を選択することも可能）
2. 選択された状態の行番号をさらにドラッグして、移動させたい場所へドロップ（移動先に赤矢印が表示されます）

未選択の状態からいきなりドラッグ＆ドロップはできません。
移動させる行をあらかじめ選択しておく必要があります。
一見面倒ですが、この仕様により複数の行をまとめて選択・移動できるようになっています。


レーダースクリーンにラベル行を追加して銘柄をグルーピングする
----

{{< image src="img-003.gif" >}}

多くの銘柄を登録する場合は、ラベル行を追加してグルーピングしておくと整理しやすくなります。

1. ラベルを挿入したい行の銘柄コードのセルを<kbd>右クリック</kbd> → <kbd><samp>ラベル行を挿入</samp></kbd> を選択
2. 「ラベル行」という名前のラベル行が追加されるので、<kbd>ダブルクリック</kbd> するか <kbd>F2</kbd> キーを押して任意の名前に変更する


レーダースクリーンにシンボルリスト（日経225銘柄など）を一度に登録する
----

複数の銘柄をリスト化したものを**シンボルリスト**と言います。
日経 225 銘柄のシンボルリストを使うと、レーダースクリーンに 225 銘柄を一度に登録できます。

1. 銘柄を挿入したい場所の銘柄コードのセルを<kbd>右クリック</kbd> → <kbd><samp>シンボルリストを挿入</samp></kbd> を選択。
2. 「シンボルリストを挿入」ダイアログから、登録したいシンボルリストを選択する（ここでは <samp>日経平均株価(225)</samp> を選択）。

{{< image src="img-004.png" >}}

3. <samp>日経平均株価 (225)</samp> というラベル名とともに 225 銘柄のシンボルが登録されます。

{{< image src="img-005.png" >}}

他にも TOPIX Core 30 などのシンボルリストが組み込みで用意されています。
