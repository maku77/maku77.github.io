パッケージを利用して GitLab をインストールした場合、GitLab 用 nginx の設定ファイルは以下の位置に配置されます。

- `/var/opt/gitlab/nginx/etc/nginx.conf`
- `/var/opt/gitlab/nginx/etc/gitlab-http.conf`
