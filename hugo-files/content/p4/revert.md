## すべてのファイルの編集状態を破棄する

`p4 opened` コマンドで表示される編集内容をすべて破棄するには、`p4 revert` コマンドを次のように実行します。

```console
$ p4 revert //...
```

次のように `-n` オプションを付けて実行すると、どのファイルが revert されるかを事前に確認できます（実際の `revert` 処理は実行されません）。

```console
$ p4 revert -n //...
```


## カレントディレクトリ以下のファイルをすべて元に戻す

```console
$ p4 revert ...
```


## edit 用に開いているファイルをすべて元に戻す

```console
$ p4 revert -a
```


## チェンジリストから編集リストを削除する方法

`p4 change` コマンドでサーバ上のチェンジリストを開き、`Files:` セクションに列挙されているファイルリストを編集することでも、ファイルのオープン状態を変更できます。
