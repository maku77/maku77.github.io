現在使用している bash のバージョンは、`$BASH_VERSION` 環境変数で参照することができます。

CentOS 5.5 の場合
----

```bash
$ echo $BASH_VERSION
3.2.25(1)-release
```

Ubuntu 12.04 の場合
----

```bash
$ echo $BASH_VERSION
4.2.24(1)-release
```
