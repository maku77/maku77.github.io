http://www.gradle.org/downloads

から gradle-1.12-bin.zip などをダウンロードして、適当なディレクトリに配置します。 gradle コマンドを実行するには、java コマンドを実行できるようになっている必要があります。 任意のバージョンの java コマンドを使用するには、JAVA_HOME 環境変数を設定しておきます。

C:\> set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_05
    （binまでは含めないことに注意）

gradle コマンドを任意のディレクトリから実行できるように、PATH を通しておきます。

C:\> set PATH=%PATH%;C:\app\gradle-1.12\bin

実行テスト:

C:\> gradle -v
