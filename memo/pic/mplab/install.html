MPLAB のインストール

  1. Mp57full.exe などを実行します。
  2. ほとんどデフォルトのまま Next ボタンで進んでいけば OK です（「Linker script Location」では Install files to MPLAB\Lkr sub directory を選択した方がよいかも）。
  3. Windows を再起動してインストール完了！
