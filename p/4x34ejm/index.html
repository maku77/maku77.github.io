Octave (MATLAB) の if ～ else の構文は以下のようになっています。

if (COND)
    ...
elseif (COND)
    ...
else
    ...
end

条件式では下記のような比較演算子が使用できます。 特に、a と b が等しくないことを表す記号が ~= であることに注意してください。

演算子意味
a == ba と b が等しい
a ~= ba と b が等しくない
a < ba が b より小さい
a > ba が b より大きい
a <= ba が b 以下
a >= ba が b 以上
aa が 0 以外

また、論理演算子を組み合わせて使用できます。

演算子意味
COND1 || COND2COND1 あるいは COND2 が真ならば真
COND1 && COND2COND1 と COND2 両方が真ならば真
!COND1真偽値の反転
